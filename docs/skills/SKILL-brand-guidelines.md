## 1. Logo

### Primary Mark
The GINZA wordmark with the kanji **銀** (silver) integrated into the letter "I", plus "Marketplace" subtitle beneath.

**Logo meaning:** Ginza is named after a famous silver coin minting district in Japan. In kanji, "GIN" (銀) means gathering place or guild, and "ZA" means silver. The logo bridges the English translation and the Japanese characters.

### Logo Files
Located in `Assets/Brand/Logos/`:
- `Black.svg` — Black logo on transparent (for light backgrounds)
- `White.svg` — White logo on transparent (for dark backgrounds)
- `Vertical.svg` — Vertical/stacked layout
- `Vertical White_1.svg` — Vertical layout, white version

### On Color Rules
Always ensure ample contrast. Approved pairings:

| Background | Logo Color |
|-----------|------------|
| White (#F8F8F8) | Black |
| Silver (#D4D4D4) | Black |
| Orange (#FF9A28) | Black |
| Black (#111111) | White |
| Tori (#AB2121) | White |
| Sky (#0378A6) | White |
| Grass (#2C4001) | White |
| Gold (#F2B705) | Black |

### Scaling
- **Minimum screen size:** 20 pixels
- **Minimum print size:** 1/4 inch
- No maximum size limit

### Clearspace
Maintain minimum clear space around the logo. Use the height of the kanji "I" element as the spacing unit. Don't crowd the logo with other elements.

### Logo Don'ts
- Don't stretch or distort
- Don't outline
- Don't use unapproved colors
- Don't apply shadows or effects
- Don't rotate
- Don't apply patterns or textures

## 2. Typography

### Font Families

**Display / Headline font: New Order**
- Used for headlines and large display text
- Weights available: Light, Regular, Medium, SemiBold, **Bold** (primary)
- Files: `Assets/Brand/Fonts/new-order-font-family/New Order *.otf`

**Body / UI font: Inter** (by Rasmus Andersson)
- Primary typeface for all body text, subheads, and UI elements
- Multi-platform compatible, optimized for legibility
- Weights available: Thin, ExtraLight, Light, Regular, Medium, SemiBold, Bold, ExtraBold, Black (+ italic variants)
- Files: `Assets/Brand/Fonts/inter/Inter-*.otf`

### Type Hierarchy

| Level | Font | Size/Leading | Case | Leading Formula |
|-------|------|-------------|------|----------------|
| **Headline** | New Order Bold | 80/88pt | Sentence case | size x 1.1 |
| **Subhead** | Inter Bold | 30/36pt | Sentence case | size x 1.2 |
| **Body** | Inter Regular | 15/21pt | Sentence case | size x 1.4 |

### Weight Hierarchy
New Order Bold (headlines) → Inter Bold (subheads) → Inter Regular (body)

### Typography Don'ts
- Don't make the subheader larger than the header
- Don't use New Order as a subheader font (it's headlines only)
- Don't over-track (too much letter spacing) or under-track text
- Don't create large or awkward spacing for headers
- Don't rotate text more than 15 degrees

## 3. Color Palette

### Primary Colors (15% each — 60% total)

| Name | HEX | RGB | Use |
|------|-----|-----|-----|
| **White** | `#F8F8F8` | 248 / 248 / 248 | Backgrounds, light mode, clean space |
| **Black** | `#111111` | 17 / 17 / 17 | Primary text, dark backgrounds |
| **Orange** | `#FF9A28` | 255 / 154 / 40 | Brand signature, primary accent, CTAs |
| **Silver** | `#D4D4D4` | 212 / 212 / 212 | Secondary backgrounds, subtle dividers |

### Secondary Colors (10% each — 40% total)

| Name | HEX | RGB | Use |
|------|-----|-----|-----|
| **Gold** | `#F2B705` | 242 / 183 / 5 | Premium indicators, accents |
| **Sky** | `#0378A6` | 3 / 120 / 166 | Links, secondary CTAs, cool accents |
| **Grass** | `#2C4001` | 44 / 64 / 1 | Nature/organic accents, subtle depth |
| **Tori** | `#AB2121` | 171 / 33 / 33 | Urgency, alerts, Japanese gate reference |

### Color Proportion
Primary colors dominate at 15% each (60% combined). Secondary colors support at 10% each (40% combined). Maintain this balance across all materials.

### Approved Color Combinations

| Background | Approved Accent Colors |
|-----------|----------------------|
| Tori (red) | Grass, Gold, Orange, Black |
| Sky (blue) | Grass, Gold, Orange, Black |
| Silver | Gold, Orange |
| White | Black, Orange |
| Black | White, Orange, Gold |
| Orange | Black, White |

**Important:** Not all colors pair well together. Always refer to this chart. When in doubt, default to Black + White + Orange — the core trio.

### Color Note
The canonical brand orange is **#FF9A28**. The daruma avatar (`Assets/ginza_ops_daruma.svg`) uses a different orange (#f87800) — that is specific to the avatar, not the brand palette.

## 5. Quick Reference for Agents

When creating any branded content (social media, Canva designs, Shopify assets, etc.):

1. **Logo:** Use `Assets/Brand/Logos/Black.svg` on light backgrounds, `White.svg` on dark
2. **Headline font:** New Order Bold (sentence case)
3. **Body font:** Inter Regular (sentence case)
4. **Primary brand color:** Orange `#FF9A28`
5. **Text color:** Black `#111111` on light, White `#F8F8F8` on dark
6. **Background defaults:** White `#F8F8F8` or Black `#111111`
7. **Accent sparingly:** Gold `#F2B705`, Sky `#0378A6`, Grass `#2C4001`, Tori `#AB2121`
8. **Tone:** See `docs/ginza-voice-SOUL.md` — smart, professional, 6/10 humor, 7/10 formality

### Social Media Dimensions (Standard)
- Instagram Post: 1080 x 1080px
- Instagram Story: 1080 x 1920px
- Facebook Post: 1200 x 630px
- X/Twitter Post: 1200 x 675px
- Discord Banner: 960 x 540px

### Do Not
- Use the logo smaller than 20px
- Stretch, rotate, outline, or add effects to the logo
- Use colors outside the approved palette
- Use New Order for body text (headlines only)
- Pair colors that aren't in the approved combinations chart
- Use the daruma orange (#f87800) as the brand orange — use #FF9A28
